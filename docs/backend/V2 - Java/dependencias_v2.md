# Dependências

### Spring Boot Starter Web

- Servidor Tomcat embutido
- Serialização/deserialização JSON
- Validação de requisições HTTP

### Spring Boot Starter Data JPA

- Gerenciamento de transações
- Mapeamento objeto-relacional
- Controle de sessões de banco de dados

### MySQL Connector

- Driver JDBC para MySQL
- Pool de conexões
- Gerenciamento de conexões com o banco

### Lombok

- Geração automática de código em tempo de compilação
- Redução de boilerplate (getters, setters, construtores)

### Spring Boot DevTools

- Hot reload durante desenvolvimento
Configurações automáticas para ambiente de desenvolvimento

### Spring Boot Starter Validation

- Validação de dados de entrada
- Anotações como @NotNull, @Size, etc.
