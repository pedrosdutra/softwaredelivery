# Como rodar o Frontend

## Pré-requisitos

- **Node.js** (>= versão 18.0)
- **npm** (>= versão 9.0)
- **Git** 


## Instalação

### 1. Instalar dependências

```bash
cd frontend
npm install
```

## Executar Localmente

```bash
npm run dev
````
